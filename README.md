## Building/Compiling
Create this .env in /server (you get the API key from [trafikverket](https://data.trafikverket.se/documentation/api-railway/get-started))
```
API_KEY=<YOUR_API_KEY>
PORT=3001
```

Then in /server run
```
npm install
```

And finally run in root
```
npm install
npm run dev
```
